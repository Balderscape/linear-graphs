Secret Patterns: Table → Equation 🧩

No graph this time — just a table of numbers hiding an equation. You’re going to learn its two secrets.

🎯 In this lesson

Given a table of x and y values, you’ll find the equation that made it. Every linear table gives itself away twice:

  • Secret 1 — the STEP. How much y changes each time x goes up by 1 — that’s m.
  • Secret 2 — the start. The value of y when x = 0 — that’s c. Read it straight off if it’s there, or step backwards to reach it.

🧩 Worked example: the pattern confesses

Here’s a table. Watch how quickly it cracks.

x01234
y 1 471013
step +3 +3 +3 +3

Secret 1: every time x goes up by 1, y jumps by +3. A steady step means a straight line — and the step is the gradient: m = 3.

Secret 2: the circled value — y when x = 0 — is sitting right there: c = 1.

y = 3x + 1

🚶‍♀️ But what if the table doesn’t start at 0?

Sneaky tables hide the x = 0 column. No problem — you just step backwards.

x12345
y5791113

Secret 1: the step is +2 every time, so m = 2.

Secret 2: the table starts at (1, 5) — but you need y at x = 0. Step back one column: undo one step of +2, so y = 5 − 2 = 3. That’s c = 3.

y = 2x + 3

Now check it against a different row: does 2 × 4 + 3 = 11? Yes ✓ — matches the table. Equation confirmed.

💡 Detective habit: always test your equation on one more row of the table. Ten seconds, and it catches nearly every slip.

🔮 Pattern machine

Deal yourself a mystery table. Work out the equation in your head (step first, then the x = 0 value), then reveal the secret.

Try these missions:
  1. Get 3 tables in a row right before revealing.
  2. Catch one with a negative step — say “m is negative” before anything else.
  3. Time yourself: can you crack a table in under 15 seconds?
💚 Easy win: these tables all show the x = 0 column, so c is a freebie — spot the step and you’re done. The drills below will take the stabilisers off.
