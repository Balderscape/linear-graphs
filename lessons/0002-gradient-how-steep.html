Gradient: How Steep? ⛰️

Every line has a steepness score. Today you learn to read it straight off the picture — no formulas, just stairs.

🎯 In this lesson

By the end you’ll glance at any straight line and read its gradient: for every 1 step right, how far does it go up? That number is m. Uphill → positive. Downhill → negative. Done.

Gradient is a staircase 🪜

Forget “steepness” as a vague vibe — make it a number. Stand on the line, take 1 step across (to the right), and ask: how far up did the line climb? That climb is the gradient. (Americans call it “slope” — same thing, different accent. 🇺🇸)

m = rise ÷ run  =  up ÷ across

Here’s y = 2x − 1 with a step-triangle drawn on it: across 1, up 2 — so m = 2. Every single step on this line climbs by 2. That’s what “gradient 2” means.

And if the line falls as you step right? Then the “up” is really a down, so the gradient is negative. Downhill left-to-right = negative m. Always. ⬇️

🎛️ The steepness dial

One pink slider, one line through the origin. Drag it and watch the staircase triangles resize themselves.

Try these missions:
  1. Make the line go downhill. What happened to the sign of m — and which way do the triangles point now?
  2. Set m = 0.5. Look at the bigger triangle: across 2, up 1. Gentle little ramp. 🛹
  3. Find what m = 0 looks like. How far “up” is each step?
💚 Easy win: you just discovered the whole story — big m = steep, small m = gentle, negative m = downhill, zero m = flat. Everything else in this lesson is just reading that off a picture.

Gradients can be fractions 🍰

What if 1 step across only climbs half a square? Awkward to see — so take 2 steps across instead. If the line goes across 2 and up 1, then each single step climbs ½:

m = rise ÷ run = 1 ÷ 2 = ½

That’s the trick for any fraction: pick a run that makes the rise land neatly on a grid corner, then divide. Rise ÷ run always works, whatever size step you take.

Lock it in 🔒

Two drills: first read gradients off ready-made triangles, then build the triangles yourself between two points. Fresh questions every replay — chase the 🌟🌟🌟.
