The Weird Ones: y = k and x = k 👻

Two lines with one-letter equations, no visible x and y, and a talent for tripping up an entire classroom. Time to make them boring.

🎯 In this lesson

You’ll handle the edge cases everyone trips on: y = 3 is a horizontal line (a floor), x = 3 is a vertical line (a wall) — and you’ll see why one of them fits y = mx + c perfectly while the other can’t be written that way at all. Plus two mind-benders: the axes themselves are lines with equations. 🤯

Flat lines: y = k 🛏️

The equation y = 3 says one thing only: “the y-coordinate is 3.” It doesn’t mention x, which means x can be anything it likes. So the points (−4, 3), (0, 3), (5, 3)… all qualify — and together they make a horizontal line, 3 above the x-axis.

And here’s the nice part: y = 3 still fits y = mx + c — it’s just y = 0x + 3 in casual clothes. Gradient m = 0 (perfectly flat), intercept c = 3. A flat line is a completely ordinary member of the family.

Walls: x = k 🧱

x = 4 says “the x-coordinate is 4” — and y can be anything. (4, −5), (4, 0), (4, 7)… stack them up and you get a vertical line: a wall standing at 4 on the x-axis.

But this one is the true outsider: a vertical line cannot be written as y = mx + c. There is no m. Its gradient is undefined — “infinitely steep”.

🧗 Why no gradient? Gradient is rise ÷ run — “up how much for each step across?” Walking up a wall, you go up any amount while going across zero. That’s rise ÷ 0, and dividing by zero is maths’ one forbidden move. So: no m, no y = mx + c, gradient undefined (not 0 — that’s the flat one!).

🤯 Plot twist: the axes are lines too

Ready for the mind-bender? The axes you’ve been drawing on all this time are themselves lines with equations:

  • The x-axis is the line y = 0 — every point on it has y-coordinate 0.
  • The y-axis is the line x = 0 — every point on it has x-coordinate 0.

Yes, that feels backwards — the x-axis has a y equation. But say it out loud: “along the x-axis, the height y is always 0” — and suddenly it’s obvious. This one turns up in exams precisely because it feels backwards. Now it’s your free mark. 😎

Never mix them up again 🧠

🧠 Memory trick for the classic mix-up: y = 3 crosses the y-axis at 3 (and is flat). x = 3 crosses the x-axis at 3 (and is a wall). The letter in the equation tells you which coordinate is locked — and the line crosses that letter’s axis at k.

👻 Wall or floor?

Slide k, then flip between the two equation types. Watch the three sample points: one coordinate is locked, the other roams free.

Try these missions:
  1. Set k = 3 and flip between y = 3 and x = 3 a few times. Floor… wall… floor… feel the difference.
  2. Slide k to 0 in each mode. Which famous lines do you land on? 🤯
  3. In y = k mode, read the sample points: which coordinate never changes as you slide?
  4. Make the wall stand at −4, then make the floor sit at −4. Say each equation before you flip.
💚 Easy win: if you can flip that toggle and predict the line before it draws, you have already beaten the most-dropped mark on this whole topic. The letter locks the coordinate — that’s the entire trick.
