Exponentials: y = bˣ 🚀

The curve behind viral videos, compound interest, and radioactive decay. It starts slow… then goes wild.

🎯 In this lesson

You’ll read y = bx — where x is now the power, not the base. You’ll see why it always passes through (0, 1), why it’s always positive, and the difference between growth 📈 and decay 📉.

Meet y = 2ˣ

This time x sits up high as the power: “raise 2 to the power x.” Each step right doubles the last y:

x−2−101234
y = 2x¼½124816

Three things to notice:

  • It always crosses the y-axis at (0, 1) — because any number to the power 0 is 1.
  • It’s always positive. Going left, y gets tinier and tinier but never reaches 0 — the x-axis is an asymptote (just like last lesson’s curve).
  • Going right, it doesn’t just rise — it accelerates. That’s exponential growth.

Growth 📈 vs Decay 📉

The base b decides everything:

  • b bigger than 1 (2, 3, 10…) → each step multiplies up → the curve grows. Doubling money, spreading rumours, populations.
  • b between 0 and 1 (½, 0.9…) → each step shrinks → the curve decays. Radioactive material, a cooling drink, medicine leaving the blood.

Growth curves sweep up to the right; decay curves slide down to the right. Both still pass through (0, 1).

🕹️ The exponential machine

Drag b. Above 1 it rockets up; below 1 it melts away.

Missions:
  1. Find the value of b that makes the curve flat. (What’s 1x always equal to?)
  2. Make the fastest-growing curve you can.
  3. Turn it into decay. What did b have to be?
  4. Whatever you pick, the curve always goes through one special point. Which one? 🎯
💚 Easy win: b > 1 grows, 0 < b < 1 decays, and every exponential passes through (0, 1).
