Amelia’s Linear Graphs Adventure · Lesson 4 of 9

Detective Mode: Graph → Equation 🕵️‍♀️

Someone drew a line and hid the equation. You’re about to learn how to unmask it — every single time.

🎯 In this lesson

So far you’ve gone equation → line. Today you go the other way: look at a drawn line and write its equation. Two clues crack every case:

  • Clue 1 — where the line crosses the y-axis gives you c.
  • Clue 2 — rise over run (how far up for each step across) gives you m.

Then you write y = mx + c. Case closed. 🔎

🗂️ Case file #1: the mystery line

Here’s a line with no equation attached. Let’s work it like a detective.

Clue 1 — the crossing point. The line cuts the y-axis at the teal dot, (0, −3). That number is c, no working needed: c = −3.

Clue 2 — rise over run. Pick any point on the line and walk the amber staircase: across 1, up 2. Gradient = rise ÷ run = 2 ÷ 1, so m = 2.

Now assemble the evidence:

y = 2x − 3

That’s the whole method. Two clues, one equation. 🕵️‍♀️

🗂️ Case file #2: the downhill suspect

One more, because downhill lines love to trick people.

Clue 1: the teal dot is at (0, 2), so c = 2. Clue 2: across 1, the line goes down 1 — and downhill means m is negative: m = −1.

y = −x + 2

The only trap in this whole topic is forgetting that minus sign. Check the tilt first, every time: uphill ⇒ positive, downhill ⇒ negative. ⛷️

🔍 Guess then check

Warm those detective muscles. Press the button, and a mystery line appears. Say its equation out loud (yes, actually out loud — it works!) before you press reveal.

Try these missions:
  1. Get 3 in a row right before revealing.
  2. Before each reveal, say which clue you found first — the dot or the tilt.
  3. Catch a downhill line and nail its negative m.
  4. Try naming c with your eyes only on the y-axis — ignore the rest of the line.
💚 Easy win: if you can name c before the reveal every time, you’re already halfway to a full equation. The gradient is just one staircase away.

Lock it in 🔒

Two drills. The first gives you both clues on a plate; the finals make you count the squares yourself. Fresh questions every replay — chase the 🌟🌟🌟.
