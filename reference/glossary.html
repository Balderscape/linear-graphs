Amelia’s Linear Graphs Adventure · Quick Reference

Glossary 📖

Every word the course uses, in plain English.

axes (singular: axis)
The two number lines that make the grid: the x-axis (horizontal) and the y-axis (vertical). Fun fact: the x-axis is the line y = 0, and the y-axis is the line x = 0.
coordinates
A point’s address, written (x, y) — across first, then up. (3, 5) means 3 across, 5 up.
coefficient
The number multiplying a letter. In y = 4x + 1, the coefficient of x is 4 — and for a line, that coefficient is the gradient.
constant
A number on its own, with no letter attached. In y = 4x + 1, the constant 1 is the y-intercept.
gradient (m)
How steep a line is: how far the line goes up for every 1 across. Uphill = positive, downhill = negative, flat = 0. Americans say slope — same thing. Between two points: m = rise ÷ run.
horizontal line
A flat line, equation y = k. Gradient 0. Every point on it has the same y-coordinate.
intersection
The point where two lines cross — the one (x, y) that satisfies both equations at once.
linear equation
An equation whose graph is a straight line — x appears on its own (no x², no √x). "Linear" literally means "line-ish".
origin
The point (0, 0), where the axes cross.
parallel
Lines with the same gradient but different intercepts. They never meet, like train tracks.
rise / run
Between two points on a line: rise is how far up (negative if down), run is how far across. Gradient = rise ÷ run.
satisfies
A point "satisfies" an equation if plugging its x and y in makes the equation true. Every point on a line satisfies the line's equation; no other point does.
simultaneous equations
Two (or more) equations that must be true at the same time. For two lines, solving them = finding the intersection.
undefined gradient
What a vertical line has. Rise ÷ run would mean dividing by zero (it goes up without going across), so no value of m works — which is why x = k can’t be written as y = mx + c.
vertical line
A wall-straight up-and-down line, equation x = k. Every point on it has the same x-coordinate. Cannot be written as y = mx + c.
y-intercept (c)
Where a line crosses the y-axis — the point (0, c). It’s the value of y when x = 0. (There’s an x-intercept too: where the line crosses the x-axis, i.e. where y = 0.)
