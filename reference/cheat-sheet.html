The equation

y = mx + c
  • m — gradient (aka slope): for every 1 across, the line goes up m. Positive = uphill ↗, negative = downhill ↘, bigger |m| = steeper. Formula between two points: m = (y₂ − y₁) ÷ (x₂ − x₁) = rise ÷ run.
  • c — y-intercept: the line crosses the y-axis at (0, c).
  • ⚠️ In y = 7 − 2x, m = −2 and c = 7. The number glued to x is always m, wherever it’s written.

Plotting y = mx + c (2 steps)

  1. Dot at (0, c) on the y-axis.
  2. From there: across 1, up m (for m = ½: across 2, up 1). Repeat, join with a ruler.

Backup method: table of values — pick x = 0, 1, 2, compute y each time, plot, join.

Graph → equation (detective recipe)

  1. c = where it crosses the y-axis.
  2. m = pick two neat grid points on the line, count rise ÷ run (downhill ⇒ negative).
  3. Write y = mx + c. Check with one point on the line.

Table → equation

  1. m = the step in y each time x goes up by 1.
  2. c = the y-value when x = 0. Not in the table? Step backwards, or use c = y − m·x with any row.
  3. Test your equation on one more row. ✓

Rearranging into y = mx + c

Mission: get y alone. Whatever you do to one side, do to the other — to every term.

  • 2y = 6x + 8 → divide everything by 2 → y = 3x + 4
  • y + 2x = 7 → subtract 2x → y = −2x + 7

Only read off m and c after y is alone.

Where two lines meet 💥

  1. Set the right-hand sides equal: m₁x + c₁ = m₂x + c₂.
  2. Solve for x. Substitute back into either equation for y.
  3. The answer is a point (x, y) — it must work in both equations.

Same m, different c ⇒ parallel, never meet. Same m and same c ⇒ same line.
